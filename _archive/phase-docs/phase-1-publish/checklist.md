## Création dans Canva Free

- [ ] Créer le template intérieur (300 DPI, marges KDP : 0.5" outside + 0.125" bleed)
- [ ] Créer 40–60 pages d'illustrations (B&W propre, lignes nettes)
- [ ] Couverture : full bleed, 300 DPI, format KDP Cover Calculator
- [ ] Exporter intérieur en PDF Press Quality (sans bleed)
- [ ] Exporter couverture en PDF Press Quality (avec bleed)

## Spécifications techniques KDP (8.5×11, 60 pages B&W)

```
Intérieur :
  - Format : 8.5" × 11"
  - Marges : Inside 0.75", Outside/Top/Bottom 0.5"
  - Résolution : 300 DPI minimum
  - Mode couleur : Grayscale ou Black & White
  - PDF : sans repères de coupe

Couverture :
  - Largeur totale = (spine width) + 2 × 8.75"
  - Spine width pour 60 pages = 60 × 0.0025" = 0.150"
  - Hauteur totale = 11.25"
  - Utiliser le KDP Cover Calculator pour les dimensions exactes
```

## Publication sur KDP

- [ ] Créer compte KDP (kdp.amazon.com)
- [ ] Nouveau livre → Paperback
- [ ] Titre : inclure les mots-clés principaux (ex: "Kawaii Food Coloring Book for Kids Ages 4-8")
- [ ] Subtitle : complément de mots-clés
- [ ] Auteur : pseudonyme ou nom réel
- [ ] Description : 2–3 paragraphes avec bénéfices + mots-clés, HTML basique autorisé (<b>, <br>)
- [ ] 7 Keywords KDP : termes de longue traîne, PAS les mots déjà dans le titre
- [ ] 2 catégories BISAC : choisir les plus spécifiques possible
- [ ] Prix : $8.99–$9.99 (US), £7.99 (UK)
- [ ] Marchés : US + UK (pas FR en v0)
